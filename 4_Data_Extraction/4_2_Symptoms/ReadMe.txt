4_2_1_1_Inner Scales_Items Coding
	量表内部，将每个题目编码为症状
4_2_1_2_Inner Scales_Items Combining
	量表内部，将测量同一个症状的题目合并，并重新赋予症状名
4_2_1_3_Inter Scales_Items Combining
	量表之间，将测量同一种症状的题目汇总在一行，从而得到所有抑郁涉及的症状
4_2_1_4_Inter Scales_Syptoms Combining
	尽管已经将测量同一种症状的题目放在了一行。但是症状的数目太多。因此，我们将类似的症状进行合并
4_2_1_5_Inter Scales_Syptoms Coding
	对每个量表下的题目所属的症状进行编码，哪些属于复合症状，哪些属于特殊症状
4_2_1_6_Inner Scales_Items Combining_DC
	重新对4_2_1_2_Inner Scales_Items Combining进行double check，检查之前所做有没有不合理的地方
4_2_1_7_Inner Scales_Items Combining
        使用4_2_1_7_Literature_Search的结果进行内容分析，即将能够归类到同一个内容的题项放在同一行。
4_2_1_8_问卷类型&症状群
        判断问卷是诊断类还是筛查类，以及判断抑郁症状可以分为几个症状群。
4_2_1_9_Measurement data
        抑郁的实证数据文件。
